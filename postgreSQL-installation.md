1. Download PostgreSQL installer : [link](https://www.postgresql.org/download/)
2. Download sesuai OS yang digunakan
3. Klik next
4. Pilih path installasi dan klik next
5. Centang yang dibutuhkan dan klik next
6. Pilih path data untuk install dan klik next
7. Masukkan password
8. Masukkan port default (5432) dan klik next
9. Pilih [Default Locale
10. Klik next
11. Klik next lagi
12. Klik Finish
13. Pilih PostgreSQL 16 dan klik next
14. Klik bagian Spatial Extensions
15. Centang PostGIS 3.4 Bundle for PostgreSQL 16 dan klik next
16. Klik next
17. Klik next lagi
18. Klik tombol "I Agree"
19. Klik next
20. Pilih path untuk install dan klik next
21. Klik finish
