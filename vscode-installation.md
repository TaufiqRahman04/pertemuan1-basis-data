Cara menginstall Visual Studio Code

1. Download VSCode installer : [link](https://code.visualstudio.com/download)
2. Download sesuai OS yang digunakan
3. Klik next
4. Centang yang dibutuhkan dan klik next
5. Klik install
6. Klik Finish
